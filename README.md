# Generative AI LSTM Next Word Prediction

## Project Overview

This project implements a Next Word Prediction model using LSTM (Long Short-Term Memory) networks.

## Dataset

COVID-19 NLP Text Classification Dataset

## Features

- Data Loading
- Text Cleaning
- Tokenization
- Vocabulary Creation
- Input-Output Pair Generation
- LSTM Model
- Next Word Prediction

## Technologies

- Python
- TensorFlow
- NumPy
- Pandas
- NLTK

## How to Run

1. Install dependencies

```bash
pip install -r requirements.txt
```

2. Open Jupyter Notebook

```
Next_Word_Prediction.ipynb
```

3. Run all cells.
